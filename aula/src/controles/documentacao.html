Documentação da API

Rota /usuario
Rota para listar usuários cadastrados.

Essa rota lista todos os usuários do banco de dados com atributos id_usuario, nome, email e senha.

Exemplo de utilização da rota /usuario:

[{"id_usuario":1,"nome":"Vinicius Aguiar","email":"vinicius@senac.com","senha":"123"},{"id_usuario":3,"nome":"admin","email":"admin@dominio.com","senha":"admin"},{"id_usuario":4,"nome":"Washington","email":"hostoluis@gmail.com","senha":"arroz123"},{"id_usuario":6,"nome":"efgh","email":"ef@gh.com","senha":"12"}]

Rota /usuario

Rota /usuario/email
Rota para buscar usuários por email.

Essa rota lista todos os usuários do banco de dados através de um email setado.

Exemplo de utilização da rota /usuario/email/vinicius@senac.com

[{"id_usuario":1,"nome":"ViniciusAguiar","email":"vinicius@senac.com","senha":"123"}]
